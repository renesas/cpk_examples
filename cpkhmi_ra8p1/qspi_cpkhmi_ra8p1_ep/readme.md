**该示例工程由 瑞萨电子-菅景梅 提供，2026年1月15日**

### 工程概述

- 该示例工程演示了基于瑞萨 FSP 的瑞萨 RA MCU 上 OSPI_B 驱动程序的基本功能。

### 支持的开发板 / 演示板：

- CPKHMI-RA8P1
  
### 硬件要求：

- 1 块 Renesas RA8 开发板：CPKHMI-RA8P1 （板上 QSPI Flash 为Winbond W25Q256JVEIQ 或者Winbond W25Q32JVSNIM）

- 1 根 USB Type A->Type C 或 Type-C->Type C 线 （支持 Type-C 2.0 即可）

### 硬件连接：

- 通过 USB Type-C 线连接 CPKHMI-RA8P1 板上的 USB 调试端口和调试用主机。

### 硬件设置注意事项：

- 无

### 软件开发环境：

- FSP 版本
  - FSP 6.2.0
- 集成开发环境和编译器：
  - e2studio v2025-10 + LLVM v18.1.3

### 第三方软件
- 无
