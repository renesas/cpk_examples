如果您需要查看RA8T2在缺省扩展板上运行的样例代码及文档，请参考 link:../cpk_ra8t2/[CPK-RA8T2]。

每个样例程序目录下都准备了对应压缩包供下载，如果您没有同步代码库的需求，可以直接下载ZIP包使用。

各个样例程序所需的软硬件配置请查看对应样例程序目录下的Readme文件。
