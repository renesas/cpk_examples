## CPKCOR-RA8D1B核心板

CPKCOR-RA8D1B核心板
是瑞萨电子为中国市场设计的模块化开发板，
使用瑞萨RA8D1MCU，核心板上已经搭载了RA8 MCU支持的较为有特色的外设接口和器件，可以直接使用核心板进行学习，评估和应用开发。

点击查看[CPKCOR-RA8D1B 核心板使用手册](../cpkcor_ra8d1b/docs/01_overview.md) 。

本目录下是存放CPKCOR-RA8D1B核心板的样例程序。
如果您没有同步代码库的需求，也可以直接下载样例程序的ZIP包使用。
